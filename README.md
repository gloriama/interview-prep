# interview-prep
Materials for software engineering job interviews

## About interviews

### Interview formats
* Onsite
  * Whiteboarding
  * Live coding
* Remote
  * Coding challenge
  * Phone screen

### Interview environments
* Whiteboard
* Phone
* Skype
* Online code environments
  * Coderpad
  * Codepen
  * JSFiddle
* Your laptop

### Types of interview questions
* Data structures / algorithms
* System design
* Facts / trivia
* Personal
* Full-stack app
* Front-end UI

## mocha-chai-suite
This suite is a plug-and-play test suite for Mocha and Chai.

### Instructions
```
cp -r mocha-chai-suite [new-directory-path]
cd [new-directory-path]
npm install
mocha .
```
